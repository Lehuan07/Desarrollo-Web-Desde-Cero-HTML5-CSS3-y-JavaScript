Html es un lenguaje de marcado que se utiliza para estructurar contenido en la web.

Css es un lenguaje de estilos que se utiliza para dar formato a la presentación de un documento HTML.

JAVASCRIPT es un lenguaje de programación que se utiliza para crear contenido interactivo en la web.

PHP es un lenguaje de programación del lado del servidor que se utiliza para crear aplicaciones web dinámicas.

PYTHON es un lenguaje de programación de propósito general que se utiliza en una variedad de aplicaciones, desde desarrollo web hasta análisis de datos.

BOOSTRAP es un framework de diseño front-end que se utiliza para crear sitios web responsivos y móviles.

JQUERY es una biblioteca de JavaScript que simplifica la manipulación del DOM y la gestión de eventos.

ANGULAR es un framework de desarrollo web que se utiliza para crear aplicaciones de una sola página.

RUBY es un lenguaje de programación que se utiliza para desarrollar aplicaciones web, especialmente con el framework Ruby on Rails.
